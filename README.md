# 광석이네 집 아카이브 템플릿

이 ZIP은 특정 웹사이트를 100% 복제한 코드가 아니라, 김광석 관련 개인 아카이브 운영 목적에 맞춰 새로 제작한 독립형 웹사이트 템플릿입니다.

## 포함 파일

- `index.html` : 메인 페이지
- `src/style.css` : 전체 디자인
- `src/app.js` : 메뉴 이동, 검색, 데모 로그인, 자료 등록 기능
- `src/firebase-config.example.js` : Firebase 연결 예시

## 실행 방법

1. ZIP 압축을 해제합니다.
2. `index.html`을 브라우저로 엽니다.
3. 데모 관리자 계정으로 로그인합니다.
   - ID: `admin`
   - PW: `demo1234`
4. `자료 등록` 버튼으로 게시글, 영상, 라디오/음성, 사진, 공지를 추가할 수 있습니다.

## 현재 기능

- 반응형 레이아웃
- 좌측 메뉴 / 모바일 햄버거 메뉴
- 검색 기능
- 게시판형 자료 목록
- 영상·음성 링크 등록
- 사진방 카드
- 공지사항
- 방명록
- 데모 로그인
- localStorage 기반 자료 저장
- Firebase 연결 준비 파일 포함

## Firebase 실제 연결 방향

현재 `app.js`는 localStorage 데모 방식입니다. 실제 운영에서는 다음 구조로 바꾸면 됩니다.

- Firebase Authentication: 관리자 로그인
- Firestore: 게시글, 영상, 음성, 공지 데이터 저장
- Firebase Storage: 사용 권한이 있는 이미지 파일 저장
- Hosting: 정적 사이트 배포
